# GLPI Audit Trail Plugin

This plugin provides a real history of changes for Tickets and TicketTasks in GLPI 11.

## Features
- Captures creation, updates, and deletions.
- Stores old and new values for modified fields.
- Tracks which user performed the change.
- Integrated "Audit Trail" tab in Ticket view.
- Supports both Ticket and TicketTask tracking.

## Installation
1. Clone or download this repository into your GLPI `plugins` directory.
2. Ensure the directory name is `audittrail`.
3. Go to **Setup > Plugins** in GLPI.
4. Install and Enable the plugin.

## Compatibility
- GLPI 11.x
